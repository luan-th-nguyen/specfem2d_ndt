75  ! NOISE_TOMOGRAPHY = 0 - turn noise tomography subroutines off; setting
76  ! NOISE_TOMOGRAPHY equal to 0, in other words, results in an earthquake
77  ! simulation rather than a noise simulation
78  !
79  ! NOISE_TOMOGRAPHY = 1 - compute "generating" wavefield and store the result;
80  ! this stored wavefield is then used to compute the "ensemble forward"
81  ! wavefield in the next noise simulation
82  !
83  ! NOISE_TOMOGRAPHY = 2 - compute "ensemble forward" wavefield and store the
84  ! result; if an adjoint simulation is planned, users need to store
85  ! seismograms (actually, cross-correlograms) for later processing
86  !
87  ! NOISE_TOMOGRAPHY = 3 - carry out adjoint simulation; users need to supply
88  ! adjoint sources constructed from cross-correlograms computed during the
89  ! "ensemble forward" step
90  !
91  ! For an explanation of terms and concepts in noise tomography, see "Tromp et
92  ! al., 2011, Noise Cross-Correlation Sensitivity Kernels, Geophysical Journal
93  ! International"
